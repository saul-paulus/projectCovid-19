Tentang

Saya, Saul Paulus, saat ini bekerja sebagai Admin Sistem Akademik pada Prodi Fisika, di Fakultas Mipa, Universitas Pattimura. ini adalah salah satu aplikasi dari beberapa aplikasi berbasis web yang telah saya kerjakan. Aplikasi web ini bertujuan untuk memberikan informasi berupa data setuasi Covid-19 di Indonesia.

untuk meningkatkan skill dan produktivitas saya terkait sistem akademik ataupun sistem informasi berbasis data maka saya mengikuti pelatihan kartu prakerja dan mengambil bidang web developer sehingga Aplikasi ini adalah sebuah project submision pada kelas Dasar-dasar fundamental front-end Developer di Dicoding.com yang dikerjakan untuk dapat lulus dari kelas tersebut. aplikasi ini dibangun dengan menggunakan tools node.js yaitu npm sebagai package mananger dan webpack sebagai kompailer untuk membuild. Sendangkan untu dapat mendisplay data setuasi covid-19 di Indonesia, digunakan AJAX (Asyncronous Javascript) yang ditulis versi vanilla Javascript untuk merequest data ke database, seperti www.kawalcorona.com atau www.covid19.go.id/peta-sebaran, selain itu sebagian code html ditulis kedalam costume Javascript sehingga membuat aplikasi web menjadi lebih ringan. kedepannya aplikasi ini akan dikembangkan untuk versi mobile dalam lingkup Progresive Web Aplication, sebagai standar Front-end Developer
